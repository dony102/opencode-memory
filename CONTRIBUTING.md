# Contributing

## Development flow

1. Install dependencies: `npm install`
2. Build locally: `npm run build`
3. Test with OpenCode MCP calls in a fresh session

## Coding rules

- Keep TypeScript strict
- Avoid runtime-only magic; prefer explicit helpers
- Keep tool responses JSON and predictable

## Pull request checklist

- Build succeeds
- Documentation updated (`README` or `docs/*`)
- No accidental changes to generated files unless intentional
